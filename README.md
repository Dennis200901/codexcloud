# Codexcloud Mini Games

A collection of simple terminal-based games implemented in Python.

## Projects

### Tic Tac Toe

A lightweight command-line tic-tac-toe implementation contained in
`tic_tac_toe.py`.

### Pacman

A terminal Pac-Man clone located in `pacman/pacman_game.py`. Use the arrow keys
or WASD to move Pac-Man, eat pellets to increase your score, and avoid the
ghosts.
